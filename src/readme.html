Инструкция по использованию SmartCalc_v2.0

Использование:

Установка приложения SmartCalc_v2.0 осуществляется при помощи Makefile и CMake файлов из папки src. Ниже приведен перечень необходимых команд.

Установка:

  1. Вызвать цель make install из папки src.
  2. Дождаться установки *.app.

Сборка в архив:

  1. Вызвать цель make dist из папки src.
  2. Архив собирается в папку src.

Удаление:

  1. Вызвать цель make uninstall из папки src.
  2. Дождаться удаления  *.app.

Запуск и использование:

  1. Для запуска приложения открыть исполняемый файл open build/SmartCalc.app/Contents/MacOS/SmartCalc, либо через терминал командой "open build/SmartCalc.app"
  2. Для вычисления: ввести выражение в поле с клавиатуры, либо кнопками и нажать на кнопку "=".
  3. Для вычисления выражения с использованием переменной x необходимо ввести в выходную строку x при помощи соответствующей кнопки, а также ввести в поле числовое значение x
  4. Для построения графика: перевести калькулятор в режим построения графика  при помощи кнопки "∟", ввести выражение и нажать  на кнопку "Graph". Для очистки поля графика использовать кнопку "Clear"

Доступные действия:

Вычисление произвольных скобочных арифметических выражений в инфиксной нотации с подстановкой значения переменной x в виде числа.

Арифметические операторы:

  1. Скобки: (a + b)
  2. Сложение: a + b
  3. Вычитание: a - b
  4. Умножение: a * b
  5. Деление: a / b
  6. Возведение в степень: a ^ b
  7. Остаток от деления: a mod b
  8. Унарный плюс: +a
  9. Унарный минус: -a

Функции:

  1. Косинус: cos(x)
  2. Синус: sin(x)
  3. Тангенс: tan(x)
  4. Арк косинус: acos(x)
  5. Арк синус: asin(x)
  6. Арк тангенс: atan(x)
  7. Квадратный корень: sqrt(x)
  8. Натуральный логарифм: ln(x)
  9. Десятичный логарифм: log(x)
